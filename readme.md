# 🧠 Next.js E-commerce Project
This project is a functional e-commerce application built using Next.js, a popular React framework for building server-side rendered and statically generated websites. The application features a user-friendly interface, authentication, and authorization, allowing users to browse products, add them to cart, and checkout securely. The project utilizes a range of technologies, including React, Redux, and Tailwind CSS, to provide a seamless user experience.

## 🚀 Features
* User authentication and authorization
* Product browsing and filtering
* Secure payment processing
* Responsive design using Tailwind CSS
* Server-side rendering and static generation using Next.js
* State management using Redux
* API calls for fetching categories and creating new products

## 🛠️ Tech Stack
* Frontend: Next.js, Redux, Tailwind CSS
* Backend: Node.js, Express.js 
* Database: PostgreSQL
